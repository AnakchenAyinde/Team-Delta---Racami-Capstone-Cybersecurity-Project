Project Overview

This capstone project was a collaborative security consultancy engagement executed by Teal Delta, a 6-member team of security professionals. We functioned as external consultants for Racami, performing an end-to-end security posture assessment that balanced technical vulnerability discovery with high-level GRC (Governance, Risk, and Compliance) strategy.

-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

Our Collective Objectives

Vulnerability Management: Our team conducted asset discovery and network-wide scans to identify critical exposures.

Compliance Mapping: We aligned Racami’s current state with the NIST Cybersecurity Framework (CSF).

Policy Development: We co-authored essential governance documentation, including an Incident Response Plan and Acceptable Use Policy (AUP).

Strategic Roadmap: We delivered a prioritized remediation plan to harden infrastructure against modern threat actors.
